# ConduGest

Sistema de gestión para escuelas de conducción.

## Descripción
ConduGest es una propuesta de software orientada a centralizar la gestión operativa y administrativa de una escuela de conducción, permitiendo controlar alumnos, clases, asistencia, recursos, reprogramaciones y solicitudes.

## Problema
Actualmente, varios procesos se gestionan de forma manual, como el control de clases, asistencia mínima, reprogramaciones, vehículos y salas psicotécnicas, lo que genera desorden y mayor carga administrativa.

## Objetivo
Desarrollar una plataforma web que permita organizar y controlar la operación diaria de la escuela de conducción.
