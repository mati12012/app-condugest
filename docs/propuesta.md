 
# Propuesta de solución

ConduGest es una plataforma web de gestión integral para escuelas de conducción. Su propósito es centralizar el control de alumnos, clases teóricas y prácticas, asistencia, profesores, vehículos, salas psicotécnicas y solicitudes, reduciendo la gestión manual y mejorando la eficiencia operativa de la escuela.
